Surviving The Storm



A small 2D sandbox game that occasionally gets interrupted by a storm.






On 23/6/2024 I made the code for this game public, I wrote it a while ago and didn't realize how poorley somethings are written. This is just a simple 2D sandbox game where you can build small structures to avoid a storm that will kill you.

Requirements and build instructions are listed in the README . It's only supported on Linux, it might work on MacOS.
